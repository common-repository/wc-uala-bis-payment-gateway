Install the module: automatically, from the “Plugins” section of WordPress.

Automatic Installation by WordPress admin
1. Access "Plugins" from the navigation side menu of your WordPress administrator.
2. Once inside Plugins, click on 'Add New' and search for 'Payment Gateway for Ualá bis on Woocommerce ' in the WordPress Plugin list
3. Click on "Install."
4. Activate it!.

Done!.

=  Configuration =

1. We will provide you with an api key and a secret key so you can enable sales.
2. From the payment settings in Woocommerce select our checkout option and click Manage to open the plugin administration screen.
3. Enter our credentials in the form, you can test that they are correct by pressing the button "verify my credentials".
4. Save your credentials and we're done! You can now start using our payment method.
